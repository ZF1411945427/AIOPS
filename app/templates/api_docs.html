API 文档

数据接入 API — 需在 Authorization 头携带 Bearer Token

POST /api/v1/metrics — 推送指标数据
curl -X POST http://localhost:8000/api/v1/metrics \
  -H "Authorization: Bearer aio_your_token_here" \
  -H "Content-Type: application/json" \
  -d '[{
    "name": "cpu_usage",
    "value": 85.5,
    "asset": "web-server-01",
    "unit": "%",
    "labels": {"host": "web-1", "region": "cn-east"},
    "timestamp": "2026-06-22T10:00:00"
  }]'
字段类型必填说明
namestring是指标名称
valuenumber是指标值
assetstring否资产名称 (自动关联)
unitstring否单位
labelsobject否标签键值对
timestampstring否ISO8601 时间 (默认当前)
POST /api/v1/events — 推送事件
curl -X POST http://localhost:8000/api/v1/events \
  -H "Authorization: Bearer aio_your_token_here" \
  -H "Content-Type: application/json" \
  -d '[{
    "cluster": "prod",
    "namespace": "default",
    "name": "web-pod-xyz",
    "kind": "Pod",
    "reason": "OOMKilling",
    "message": "Pod was killed due to memory limit",
    "severity": "critical"
  }]'
GET /api/v1/query/metrics — 查询指标
curl "http://localhost:8000/api/v1/query/metrics?name=cpu_usage&hours=2&limit=50" \
  -H "Authorization: Bearer aio_your_token_here"
参数类型说明
namestring指标名称 (可选)
assetstring资产名称 (可选)
hoursint查询最近 N 小时数据 (默认1)
limitint最大返回条数 (默认100)
Token 权限说明
权限可访问接口
readGET /api/v1/query/*
writePOST /api/v1/metrics, POST /api/v1/events
admin所有接口
