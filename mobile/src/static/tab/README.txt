tabBar 图标占位说明
====================

本目录需要放入 8 个 PNG 图标文件，规格：81x81 像素，透明背景。

所需文件清单：
- home.png       首页（未选中）灰色 #7A8898
- home_on.png    首页（选中）蓝色 #3B82F6
- alert.png      告警（未选中）灰色
- alert_on.png   告警（选中）蓝色
- ai.png         AI 助手（未选中）灰色
- ai_on.png      AI 助手（选中）蓝色
- mine.png       我的（未选中）灰色
- mine_on.png    我的（选中）蓝色

设计建议：
- 使用简洁的线性图标（line icon）风格
- 未选中态颜色 #7A8898（text-muted）
- 选中态颜色 #3B82F6（primary）
- 图标尺寸 81x81 px，PNG 格式，透明背景
- 推荐使用 Iconfont / Tabler Icons / Phosphor Icons 等图标库导出

未替换前：UniApp 在缺少图标时会显示空白 tabBar，不影响功能运行，
但建议尽快替换为正式图标以获得完整视觉效果。
